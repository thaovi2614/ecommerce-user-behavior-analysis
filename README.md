# Ecommerce User Behavior Analysis

## Project Overview
Phân tích hành vi khách hàng ecommerce bằng Python nhằm khám phá:
- xu hướng doanh thu
- hành vi mua hàng
- retention rate
- customer segmentation bằng RFM analysis

## Dataset
Online Retail Dataset

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Main Analysis
### 1. Revenue Trend Analysis
Phân tích xu hướng doanh thu theo thời gian.

### 2. Top Countries by Revenue
Xác định quốc gia mang lại doanh thu cao nhất.

### 3. Top Products Analysis
Phân tích sản phẩm tạo doanh thu lớn nhất.

### 4. Customer Segmentation (RFM)
Phân nhóm khách hàng:
- VIP Customers
- Loyal Customers
- At Risk Customers
- Regular Customers

### 5. Cohort Retention Analysis
Phân tích khả năng giữ chân khách hàng theo thời gian.

## Key Insights
- United Kingdom tạo phần lớn doanh thu.
- Một nhóm nhỏ khách hàng VIP đóng góp doanh thu đáng kể.
- Retention rate giảm mạnh sau những tháng đầu tiên.
- Doanh nghiệp cần tập trung giữ chân khách hàng trung thành.
